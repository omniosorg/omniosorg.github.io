The Open Source Enterprise Server OS with just the features you need.

Key Features

Outstanding storage platform

ZFS, OmniOS' native file-system, combines a file-system and volume manager with strong data-integrity protection. Easily share volumes via iSCSI, CIFS and NFS.

Lightweight virtualisation

Run lightweight OmniOS or Linux virtual machines in containers without the overhead of a traditional hypervisor and with full resource control.

Hardware virtualisation

OmniOS also provides full hardware virtualisation via KVM for running guests such as Microsoft Windows, FreeeBSD and many others.

Software-defined networking

Virtualise your network infrastructure through virtual interfaces and switches and even allocate dedicated resources for specific services.

Built-in tracing

DTrace provides a comprehensive dynamic tracing framework to aid troubleshooting across the whole software stack. Always available, instrument anything on demand.

Open Source

OmniOS is open-source, self-hosting and maintained on GitHub. All development is done out in the open with pull-requests and anyone can grab the source and build OmniOS themselves.

Speedy deployment

OmniOS' revolutionary installation framework - Kayak - is blindingly fast, whether installing 100s of servers over the network or just building a single VM (in < 2 minutes).

Seamless updates and upgrades

Easily manage installed packages with digitally-signed updates. LTS releases receive updates for four years so you can schedule upgrades when convenient.

Commercial support

Commercial support packages are available to provide custom assistance and direct access to the OmniOS developers in case of any problems
